# Service API

**Service API** — Laravel-приложение с двумя основными компонентами: 
веб-сайт с авторизацией через Google и отображением погоды,
а также REST API для работы с пользователями и данными.

---

## Веб-сайт

Веб-часть приложения предоставляет:

- Авторизация через Google (OAuth, Laravel Socialite)  
- Получение и отображение данных внешнего API погоды  
- Личный кабинет для авторизованных пользователей

---

## REST API

API предоставляет функциональность для работы с пользователями:

- Регистрация  
- Авторизация (логин/пароль)  
- Получение токена  
- Верификация почты  
- Сброс и восстановление пароля  
- Доступ к пользовательским данным
